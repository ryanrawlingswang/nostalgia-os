# 🪟 NostalgiaOS

> *"It's giving Windows XP" - every millennial ever*

Travel back to 2001 with this vibe-coded portfolio generator. Remember when life was simple, your biggest worry was AIM away messages, and Windows XP was the coolest thing ever? We do too. That's why we created NostalgiaOS - a portfolio generator that's all about that Windows XP aesthetic.

![Windows XP Vibe](https://i.imgur.com/7Y9DNxt.png)

## ✨ What's the Vibe?

This isn't your regular portfolio generator. This is a full-on time machine that:

- 🌅 Rocks that iconic Bliss wallpaper (you know the one)
- 🎨 Serves Windows XP realness with every click
- 🪟 Gives you those satisfying window dragging animations
- 🔵 Makes everything blue (the good kind of blue, not the 🟦 of death)
- 📝 Uses AI to generate your portfolio (because we're not *completely* stuck in 2001)
- 🔄 Lets you view past generations (like browsing your old MySpace profiles)

## 🛠️ Built With Vibes

- Next.js 14 (because we're modern like that)
- TypeScript (for when you miss those Windows error messages)
- Tailwind CSS (making Windows XP look good in 2024)
- Shadcn/ui (but make it XP)

## 🚀 Start Your Nostalgia Trip

1. Clone this bad boy:
```bash
git clone https://github.com/ryanrawlingswang/nostalgia-os.git
```

2. Install the goods:
```bash
pnpm install # npm is so Windows Vista
```

3. Fire it up:
```bash
pnpm dev # like double-clicking on Internet Explorer, but better
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser (Chrome is fine, we won't judge)
